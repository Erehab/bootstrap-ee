Bootstrap Theme Colors

BSEE overrides for Bootstrap's semantic color system. Generates .btn-*, .bg-*, .text-*, .badge, .alert-* etc. automatically.

Name SCSS Variable Hex FBS Source Usage

success $success #2abb7f $fbs-green-500 Positive status, confirmed actions
warning $warning #faa53d $fbs-orange-400 Caution — use as bg/badge, low contrast for text
danger $danger #ef5c48 $fbs-red-500 Errors, destructive actions
info $info #1d7afc $fbs-blue-600 Links, informational, card titles
