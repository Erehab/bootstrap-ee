Font Awesome: Old (4.x) vs New (6 Pro)

Legacy used FA 4.x — a single webfont with one weight (solid), loaded via .fa prefix. BSEE bundles FA Pro 6 — six styles (solid, regular, light, thin, duotone, sharp), loaded via .fa-solid / .fas etc. FA4 is loaded via CDN in the left column for comparison; FA6 Pro is served from css/bs-ee.css on the right.

Legacy — Font Awesome 4.x
Usage Pattern
<i class="fa fa-home"></i>
<i class="fa fa-user"></i>
<i class="fa fa-check"></i>
<i class="fa fa-times"></i>
Single .fa prefix. One style only (solid).
⚠ Many FA4 icon names changed in FA5/FA6. Templates using .fa-times, .fa-pencil, .fa-home etc. need class updates.
New — Font Awesome 6 Pro (bundled)
Usage Pattern
<i class="fa-solid fa-house"></i>        <!-- solid -->
<i class="fa-regular fa-house"></i>      <!-- regular -->
<i class="fa-light fa-house"></i>        <!-- light -->
<i class="fa-thin fa-house"></i>         <!-- thin -->
<i class="fa-duotone fa-house"></i>      <!-- duotone -->
Style-first prefix. Legacy shorthand still works: .fas = solid, .far = regular, .fal = light, .fat = thin, .fad = duotone.
