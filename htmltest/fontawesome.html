Icons

Bootstrap EE bundles Font Awesome Pro 6 — six styles, thousands of icons. No separate script or kit required.

How to Find Icons

Search fontawesome.com

The official icon search at fontawesome.com/icons lists every icon with its exact class name. Filter by style (solid, regular, light, thin, duotone, sharp).

fontawesome.com/icons?f=classic&s=solid
Naming Tips
  • Icon names use kebab-case: fa-magnifying-glass
  • When unsure, try common words first: fa-house, fa-gear
  • Medical terms often exist: fa-stethoscope, fa-tooth, fa-syringe
  • Many FA4 names still work via the compatibility shim
  • Prefer light/thin weight for decorative UI; solid for functional

Basic Usage

<!-- Solid (default — use for UI actions) -->
<i class="fa-solid fa-house"></i>

<!-- Regular (medium weight) -->
<i class="fa-regular fa-envelope"></i>

<!-- Light (thin strokes — decorative) -->
<i class="fa-light fa-star"></i>

<!-- Thin (thinnest — large sizes only) -->
<i class="fa-thin fa-calendar"></i>

<!-- Duotone (two-color — Pro only) -->
<i class="fa-duotone fa-house"></i>

<!-- Sharp solid (squared corners — Pro only) -->
<i class="fa-sharp fa-solid fa-house"></i>

<!-- Legacy shorthand aliases still work -->
<i class="fas fa-house"></i>   <!-- solid -->
<i class="far fa-house"></i>   <!-- regular -->
<i class="fal fa-house"></i>   <!-- light -->
<i class="fat fa-house"></i>   <!-- thin -->
<i class="fad fa-house"></i>   <!-- duotone -->

Six Styles

Same icon, six weights. Pro-only styles require the FA Pro kit bundled in BSEE.

fa-solid .fas
fa-regular .far
fa-light .fal — Pro
fa-thin .fat — Pro
fa-duotone .fad — Pro
fa-sharp + fa-solid — Pro
Use solid for interactive/action icons (buttons, nav). Use light or thin for decorative or large display icons.

Sizing

Use FA size classes or set font-size directly in CSS/style.

fa-xs fa-sm (base) fa-lg fa-xl fa-2xl fa-2x fa-3x fa-4x 
<!-- FA size classes -->
<i class="fa-solid fa-star fa-xs"></i>      <!-- 0.75em -->
<i class="fa-solid fa-star fa-sm"></i>      <!-- 0.875em -->
<i class="fa-solid fa-star fa-lg"></i>      <!-- 1.25em -->
<i class="fa-solid fa-star fa-xl"></i>      <!-- 1.5em -->
<i class="fa-solid fa-star fa-2xl"></i>     <!-- 2em -->
<i class="fa-solid fa-star fa-2x"></i>      <!-- 2em (same as 2xl) -->
<i class="fa-solid fa-star fa-3x"></i>      <!-- 3em -->

<!-- Or just use CSS -->
<i class="fa-solid fa-star" style="font-size: 2rem;"></i>

Animations

Font Awesome 6 includes built-in animation classes. Add them alongside the icon class.

fa-spin
fa-spin-reverse
fa-spin-pulse
fa-beat
fa-shake
fa-bounce
fa-flip
fa-beat-fade
fa-fade 
<!-- Basic spin -->
<i class="fa-solid fa-rotate fa-spin"></i>

<!-- Reverse spin -->
<i class="fa-solid fa-rotate fa-spin fa-spin-reverse"></i>

<!-- Pulse/stepped spin (loading indicator) -->
<i class="fa-solid fa-rotate fa-spin fa-spin-pulse"></i>

<!-- Beat (scale pulse) -->
<i class="fa-solid fa-heart fa-beat"></i>

<!-- Beat + fade combined -->
<i class="fa-solid fa-heart fa-beat-fade"></i>

<!-- Shake -->
<i class="fa-solid fa-bell fa-shake"></i>

<!-- Bounce -->
<i class="fa-solid fa-triangle-exclamation fa-bounce"></i>

<!-- Flip (horizontal by default) -->
<i class="fa-solid fa-star fa-flip"></i>

<!-- Fade -->
<i class="fa-solid fa-circle fa-fade"></i>
CSS Custom Properties for Animations
Property Default Applies to
--fa-animation-duration1sAll animations
--fa-animation-delay0sAll animations
--fa-animation-iteration-countinfiniteAll animations
--fa-animation-directionnormalAll animations
--fa-beat-scale1.25fa-beat, fa-beat-fade
--fa-bounce-height-0.5emfa-bounce
--fa-flip-x, --fa-flip-y1 / 0fa-flip
 
<!-- Slow spin with custom duration -->
<i class="fa-solid fa-rotate fa-spin"
   style="--fa-animation-duration: 3s;"></i>

<!-- Bigger beat -->
<i class="fa-solid fa-heart fa-beat"
   style="--fa-beat-scale: 1.5;"></i>

<!-- Vertical flip -->
<i class="fa-solid fa-star fa-flip"
   style="--fa-flip-x: 0; --fa-flip-y: 1;"></i>

FA4 → FA6 Name Changes

Many icon names changed between FA4 and FA6. The compatibility shim allows old names to work temporarily, but templates should be migrated to FA6 names.

fa-home→fa-house fa-times→fa-xmark fa-cog→fa-gear fa-search→fa-magnifying-glass fa-sign-out→fa-right-from-bracket fa-sign-in→fa-right-to-bracket fa-pencil-square-o→fa-pen-to-square fa-trash-o→fa-trash-can
fa-picture-o→fa-image fa-exclamation-triangle→fa-triangle-exclamation fa-info-circle→fa-circle-info fa-check-circle→fa-circle-check fa-times-circle→fa-circle-xmark fa-bar-chart→fa-chart-bar fa-bar-chart-o→fa-chart-bar fa-area-chart→fa-chart-area
The FA6 compatibility shim maps many FA4/FA5 class names automatically, but use correct FA6 names in new code.
