How to Use

In CSS / SCSS
.my-element {
  color: var(--ds-text-brand);
  background: var(--ds-background-brand-subtlest);
  border-color: var(--ds-border-brand);
}
In SCSS (via $ds-* variables)
.my-element {
  color: $ds-text-brand;
  background: $ds-background-brand-subtlest;
  border-color: $ds-border-brand;
}
